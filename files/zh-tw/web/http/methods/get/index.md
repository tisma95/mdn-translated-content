**HTTP `GET` method** 方法請求展示指定資源。 使用 `GET` 的請求只應用於取得資料。

| Request 有 body            | No  |
| -------------------------- | --- |
| 成功的 response 有 body    | Yes |
| {{Glossary("Safe")}}       | Yes |
| {{Glossary("Idempotent")}} | Yes |
| {{Glossary("Cacheable")}}  | Yes |
| 允許在 HTML 表單           | Yes |

## 格式

```plain
GET /index.html
```
