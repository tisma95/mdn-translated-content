**`504 Gateway Timeout`** 錯誤表明伺服器以閘道器或代理訪問時，並沒有上游伺服器即時收到完成請求所需的回應。

> **備註：** [閘道器](https://zh.wikipedia.org/wiki/閘道器)可能位於網路上的不同地方。502 錯誤通常也不是開發者可以修復的，通常需要在要訪問的的伺服器或代理修復之。

## 狀態

```plain
504 Gateway Timeout
```
