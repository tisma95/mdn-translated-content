**ASCII** (_American Standard Code for Information Interchange，美国信息交换标准码_) 是计算机中最常用的编码方式，用于将字母，数字，标点符号和控制字符转换为计算机可以理解的数字形式。从 2007 年开始逐渐被{{Glossary("UTF-8")}} 代替。
