## 概述

The **`title`** property of the {{domxref("StyleSheet")}} interface returns the advisory title of the current style sheet.

The title is often specified in the {{domxref("StyleSheet/OwnerNode", "ownerNode")}}.

## 注意

The title is often specified in the {{domxref("StyleSheet/OwnerNode", "ownerNode")}}.
