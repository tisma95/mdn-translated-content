{{APIRef("HTML DOM")}}

**`HTMLMediaElement.autoplay`** 属性等同于 HTML 属性中的 [`autoplay`](/zh-CN/docs/Web/HTML/Element/video#autoplay), 它表示当 player 下载到足够可以播放的媒体文件时，是否自动播放视频。

注意：一些版本的 Chrome 只支持**`autostart`**, 不支持**`autoplay`**
