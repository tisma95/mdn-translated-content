{{domxref("VisualViewport")}} 接口的只读属性 **`offsetTop`** 返回视觉视口的顶部相对于布局视口的顶部的偏移量，以 CSS 像素为单位。

## 语法

```plain
var offsetTop = VisualViewport.offsetTop
```

### 值

一个双精度值。
