### 概述

返回当前文档的 Content-Type(MIME) 类型。

### 语法

```plain
contentType = document.contentType;
```

该属性为只读。

### 备注

该属性的返回值是浏览器检测到的，不一定是直接读取 HTTP 响应头中的或者 HTML 中 meta 标签指定的值。

### 规范

非标准属性，仅[Gecko](/zh-CN/Gecko)支持。
