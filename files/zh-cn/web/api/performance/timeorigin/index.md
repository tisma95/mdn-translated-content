接口 {{domxref("Performance")}} 的只读属性 **`timeOrigin`** 返回一个表示 the performance measurement 开始时间的高精度 timestamp

## 语法

```plain
var timeOrigin = performance.timeOrigin
```

### 值

高精度 timestamp，例如 1518354643295.86
