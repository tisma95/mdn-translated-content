## 语法

```plain
timingInfo = performance.memory
```

## 属性

- `jsHeapSizeLimit`
  - : 上下文内可用堆的最大体积，以字节计算。
- `totalJSHeapSize`
  - : 已分配的堆体积，以字节计算。
- `usedJSHeapSize`
  - : 当前 JS 堆活跃段（segment）的体积，以字节计算。
