{{APIRef("DOM")}}

{{domxref("NamedNodeMap")}} 接口的 **`NamedNodeMap.getNamedItem()`** 方法返回对应给定名称的{{domxref("Attr", "属性")}}，如果没有对应名称的属性则返回 `null`

## 语法

```plain
myAttr = attrs.getNamedItem(name)
```

### 参数

- `name` 是期望获取的属性名
