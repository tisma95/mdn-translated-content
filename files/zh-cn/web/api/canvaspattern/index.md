{{APIRef("Canvas")}}

**`CanvasPattern`** 接口表示描述一个模板（基于 image、canvas 或 video）的不透明对象，通过 {{domxref("CanvasRenderingContext2D.createPattern()")}} 方法创建。

## 属性

_非透明对象，没有暴露的属性。_

## 方法

_没有继承来的方法。_

- {{domxref("CanvasPattern.setTransform()")}}
  - : 应用 {{domxref("DOMMatrix")}} 对模板做线性变换
