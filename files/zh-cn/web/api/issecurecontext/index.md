全局只读属性 **`isSecureContext`** 返回一个布尔值，指示当前[上下文是安全的](/zh-CN/docs/Web/Security/Secure_Contexts)（`true`）还是不安全的（`false`）。

## 值

一个布尔值，`true` 或 `false`。
