### 概述

返回触点相对于屏幕上边沿的 Y 坐标。不包含页面滚动的偏移量。

### 语法

```plain
var y = touchItem.screenY;
```

### 返回值

- `y`
  - : 触点相对于屏幕上边沿的 Y 坐标。不包含页面滚动的偏移量。

### 标准定义

[Touch Events Specification](http://www.w3.org/TR/touch-events/)
