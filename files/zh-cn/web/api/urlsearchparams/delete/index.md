**delete()** 是 {{domxref("URLSearchParams")}} 接口的一个方法。可以删除指定名称的所有搜索参数。

## 语法

```plain
URLSearchParams.delete(name)
```

### 参数

- name
  - : 需要删除的键值名称

### 返回

无
