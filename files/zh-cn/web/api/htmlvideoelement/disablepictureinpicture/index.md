{{APIRef("HTML DOM")}}

{{domxref("HTMLVideoElement")}} 提供的 **`disablePictureInPicture`** HTML 属性用于指示是否应该向用户建议画中画的功能，或者自动请求该功能。

## 属性

一个布尔值，如果为`true`,阻止用户代理向用户建议该功能。
