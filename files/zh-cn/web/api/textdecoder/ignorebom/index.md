{{APIRef("Encoding API")}}

{{domxref("TextDecoder")}} 接口的只读属性 **`ignoreBOM`** 是一个{{jsxref('Boolean', '布尔值', '', '1')}}，表示是否忽略[字节顺序标记](https://www.w3.org/International/questions/qa-byte-order-mark)。

## 值

如果字节顺序标记被忽略，则是 `true`；否则是 `false`。
