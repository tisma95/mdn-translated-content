{{apiref("TextEncoder")}}

**`TextEncoder()`** 构造函数总是返回一个新创建的 {{domxref("TextEncoder")}} 对象，该对象将生成具有 UTF-8 编码的字节流。

## 语法

```js-nolint
new TextEncoder();
```

### 参数

无。
