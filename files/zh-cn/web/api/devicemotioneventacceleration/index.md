设备加速对象可以提供有关设备沿三个轴（x、y、z）的加速度的信息。

## 属性

- {{domxref("DeviceAcceleration.x")}} {{readonlyInline}}
  - : 沿 X 轴的加速度量（只读）
- {{domxref("DeviceAcceleration.y")}} {{readonlyInline}}
  - : 沿 Y 轴的加速度量（只读）
- {{domxref("DeviceAcceleration.z")}} {{readonlyInline}}
  - : 沿 Z 轴的加速度量（只读）
