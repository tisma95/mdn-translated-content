**`window.event`** 是一个由微软 IE 引入的属性，只有当 DOM 事件处理程序被调用的时候会被用到。它的值是当前正在处理的事件对象。

## 规范

没有任何规范。

微软[在 MSDN 上有相关介绍](<https://msdn.microsoft.com/en-us/library/ms535863(v=vs.85).aspx>)。
