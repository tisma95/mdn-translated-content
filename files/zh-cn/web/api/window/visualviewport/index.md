{{domxref("Window")}} 接口的 visualViewport 只读属性返回一个{{domxref("VisualViewport")}}对象，该对象表示给定窗口的可视视口。

## 语法

```plain
var visualViewport = Window.visualViewport
```

### 值

一个 {{domxref("VisualViewport")}} 对象。
