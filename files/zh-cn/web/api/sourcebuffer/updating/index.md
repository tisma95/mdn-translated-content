{{domxref("SourceBuffer")}} 接口的只读属性 **`updating`** 表示 `SourceBuffer` 目前是否正在更新——即当前是否有 {{domxref("SourceBuffer.appendBuffer()")}}、{{domxref("SourceBuffer.appendStream()")}} 或 {{domxref("SourceBuffer.remove()")}} 操作正在进行。

## 值

一个布尔值。
