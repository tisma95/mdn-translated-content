{{APIRef("Intersection Observer API")}}{{domxref("IntersectionObserver")}} 的`disconnect()`方法终止对所有目标元素可见性变化的观察。

## 语法

```js
intersectionObserver.disconnect();
```

### 参数

None.

### 返回值

`undefined`.
