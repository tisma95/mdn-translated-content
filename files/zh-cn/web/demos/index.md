Mozilla 支持各种令人兴奋的开源 Web 技术，我们鼓励大家使用它们。此页面提供了有关这些技术的一些有趣演示链接。

如果你知道开源 Web 技术的优秀演示或者应用，就在这里（以及 [英文页面](/zh-CN/docs/Web/Demos_of_open_web_technologies)）添加一个合适的链接吧。
