HTTP **`418 I'm a teapot`** 客户端错误响应代码表示服务器拒绝冲泡咖啡，因为它是个茶壶。

该错误是超文本咖啡壶控制协议的参考，和 1998 年愚人节的玩笑。

## 状态

```plain
418 I'm a teapot
```
