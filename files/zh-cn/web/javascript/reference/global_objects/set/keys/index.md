**`keys()`** 方法是 [`values()`](/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Set/values) 方法的别名。

## 语法

```js-nolint
keys()
```

### 返回值

一个新的迭代器对象，按插入顺序包含给定 `Set` 中每个元素的值。
