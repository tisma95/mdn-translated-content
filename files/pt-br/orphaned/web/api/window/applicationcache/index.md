## Sumário

Retorna uma referência para o objeto de cache da janela.

## Sintaxe

```
cache = window.applicationCache
```

### Parâmetros

- `cache` é uma referência de objeto para {{domxref("OfflineResourceList")}}.

## Especificação

Não faz parte de denhuma especificação.
