**`DOMRect()`** コンストラクターは、新しい {{domxref("DOMRect")}} オブジェクトを作成します。

## 構文

```js
new DOMRect(x, y, width, height);
```

### 引数

- `x`
  - : この `DOMRect` の原点の `x` 座標です。
- `y`
  - : この `DOMRect` の原点の `y` 座標です。
- `width`
  - : この `DOMRect` の幅です。
- `height`
  - : この `DOMRect` の高さです。

### 返値

新しい {{domxref("DOMRect")}} オブジェクトです。
