**`MediaStreamTrack.id`** は読み取り専用のプロパティであり、このトラックのための固有識別子 (GUID)を含む文字列を返します。この値は{{glossary("user agent", "ユーザーエージェント")}}によって生成されます。

## 構文

```js
const id = track.id;
```
