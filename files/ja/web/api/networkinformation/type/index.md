{{apiref("Network Information API")}}{{SeeCompatTable}}

**`NetworkInformation.type`** は読み取り専用プロパティで、端末がネットワーク通信に使用している接続の種類を返します。

{{AvailableInWorkers}}

### 値

次の値のいずれかである列挙値です。

- `"bluetooth"`
- `"cellular"`
- `"ethernet"`
- `"none`"
- `"wifi"`
- `"wimax"`
- `"other"`
- `"unknown"`
