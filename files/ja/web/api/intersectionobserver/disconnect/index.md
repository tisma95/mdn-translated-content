{{APIRef("Intersection Observer API")}}

{{domxref("IntersectionObserver")}} の **`disconnect()`** メソッドは、すべての対象要素について、表示状態の変化の監視を停止します。

## 構文

```js-nolint
disconnect()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。
