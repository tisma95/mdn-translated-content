- [XSL Results Firefox extension](https://addons.mozilla.org/ja/firefox/addon/xsl-results/) (現在レビュー待ち) - XML 文書 (現在ロードされている文書または手動で入力/貼り付けられた文書) に XSL スタイルシート (URL またはファイルシステムで手動で入力されたもの) を適用して、XSL を試すことができます。
